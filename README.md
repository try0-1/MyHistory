### デモ用プロジェクションマッピングソフトウェア(Repository非公開)
https://youtu.be/eTSmJGXVvK8

- 概要:前述の学会デモの為のソフトウェア。頂点位置の保存、頂点のナビゲータの表示切替、歪みのアニメーション切り替え、マテリアルの変更、輝度補正が可能である。輝度補正はwebcameraを使い、3面のそれぞれのRGB値の平均を輝度とし、3面のうち輝度が最小の面（暗い面）の輝度を基準として残り二つの面の輝度を下げることで輝度補正を実現した。輝度補正することで3面の明るさが平均化され、輝度の違いによる違和感をなくすことができた。

- 使用機材:Unity,Projector,Web camera
